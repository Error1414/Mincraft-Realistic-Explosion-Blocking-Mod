# Mincraft-Realistic-Explosion-Blocking-Mod
A lightweight pack for realism to prevent you from being bombarded through walls

project page: **Coming**

# **Tutorial**

**Manual Installation:**

Download the file and place it in a curseforge profiles datapack folder here: 

C:\Users\your_name\curseforge\minecraft\Instances\your_profile_name\datapacks

Curseforge installation: https://www.youtube.com/watch?v=zDINExUM6do&t=10s

Automatic Installation:
1. After downloading curseforge open it and select mincraft in the select game tab.
2. Create a new profile by pressing the "create+" button and choose a version of you're liking between 1.20 - 1.21.
3. Search upp "Realistic Explosion Blocking" in the datapack tab while in the profile you just created, press install right next to the datapack called "Realistic Explosion Blocking".
4. Now you can just press play and enjoy the game!
